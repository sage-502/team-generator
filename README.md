# Random Team Generator

* 운영의 귀찮음을 덜기 위한 미니 프로그램
* CSV 기반 랜덤 팀 생성 프로그램 (Python GUI)

※ ChatGPT가 작성함.

---

## 기능
- CSV 파일 입력 (이름, 분야)
- 논문 / 워게임 분야 분리
- 3~5명 랜덤 팀 생성
- 5명 이하인 경우 한 팀
- 랜덤 시드: 현재 시각
- GUI 기반 실행

---

## 사용 기술
- Python 3
- tkinter
- csv
- random

---

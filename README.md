# Random Team Generator

공강 5시간 동안 심심해서 만든 **랜덤 팀 생성기**입니다.</br>
CSV 파일을 입력하면 분야별로 인원을 분류하고, **3~5명 단위의 랜덤 팀**을 생성합니다.

랜덤 시드와 생성 시각을 함께 출력하여 **재현 가능성**과 **투명성**을 확보했습니다.

GUI는 `Tkinter`를 사용하며, 결과는 화면 출력 / CSV 저장 / 클립보드 복사로 활용할 수 있습니다.

---

## Features

* CSV 파일 입력 (이름, 분야)
* 분야별 인원 자동 분류
* **3~5명 균등 랜덤 팀 생성**
* 랜덤 시드 기반 결과 재현 가능
* 생성 시각 및 시도 횟수 기록
* 결과 CSV 파일 저장
* 결과 클립보드 복사
* 간단한 Tkinter GUI

---

## Input Format

CSV 파일은 다음 형식을 사용합니다.

```
이름,분야
홍길동,논문
김철수,워게임
이영희,워게임
```

* 1열: 이름
* 2열: 분야 (`논문` / `워게임`)

---

## Team Generation Rules

각 분야는 **독립적으로 팀이 생성됩니다.**

* 인원이 **5명 이하**

  * 전체를 하나의 팀으로 구성

* 인원이 **5명 초과**

  * **3~5명 단위의 랜덤 팀**으로 분배
  * 팀 수는 자동 계산

예시:

```
논문1팀 : A, B, C, D
워게임1팀 : E, F, G, H
워게임2팀 : I, J, K
```

## How to Run

```bash
python team_generator.py
```
혹은 그냥 IDLE에서 실행.

GUI가 실행되면:

1. CSV 파일 선택
2. 팀 생성
3. 결과 확인
4. CSV 저장 또는 클립보드 복사
